Gitスタッシュの使い方

Gitスタッシュは、作業中の変更を一時的に保存するための機能です。以下に基本的な使い方を示します。

1. 変更をスタッシュする

git stash

このコマンドを実行すると、作業中の変更がスタッシュに保存され、ワーキングディレクトリがクリーンな状態になります。

2. スタッシュを一覧表示する

git stash list

現在のスタッシュの一覧を表示します。

3. スタッシュを適用する

git stash apply

最新のスタッシュを適用します。特定のスタッシュを適用する場合は、git stash apply stash@{n}のように指定します。

4. スタッシュを削除する

git stash drop

最新のスタッシュを削除します。特定のスタッシュを削除する場合は、git stash drop stash@{n}のように指定します。

5. スタッシュをポップする

git stash pop

最新のスタッシュを適用し、同時にスタッシュから削除します。
